⚡ Quick ADB Commands

Device Info

Model

Get device model name

adb shell getprop ro.product.model

Android Version

OS version and API level

adb shell getprop ro.build.version.release

Storage

Check available storage

adb shell df -h /data

Apps & Permissions

List Apps

Show all user-installed apps

adb shell pm list packages -3

Revoke Permission

Remove dangerous permission

adb shell pm revoke pkg perm

Uninstall App

Remove system app for user

adb shell pm uninstall -k --user 0 pkg

Settings & System

Disable Ad Tracking

Limit advertising ID tracking

adb shell settings put global limit_ad_tracking 1

Turn Off Location

Disable all location services

adb shell settings put secure location_mode 0

Screenshot

Take and save screenshot

adb exec-out screencap -p > screen.png
