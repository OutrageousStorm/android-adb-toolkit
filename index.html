🤖 Android ADB Toolkit

Web-based Android debugging and control via ADB.

Quick Commands

Device Info

adb shell getprop ro.product.model

Install APK

adb install app.apk

Screen Capture

adb exec-out screencap -p > screen.png

Reboot to Recovery

adb reboot recovery

List Packages

adb shell pm list packages

Grant Permission

adb shell pm grant com.app permission.NAME
