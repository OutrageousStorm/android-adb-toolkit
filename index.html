🤖 Android ADB Toolkit

📱 Device Management
📋

Device Info

Get complete device details: model, Android version, CPU, RAM, storage, battery, network.

adb shell getprop ro.product.model
📦

App Management

List, install, uninstall, or disable apps. Bulk operations supported.

adb shell pm list packages
💾

Backup & Restore

Backup APKs and data. Restore everything with one command.

adb pull /data/app/
🔒 Security & Privacy
🔐

Permission Audit

Scan every app for dangerous permissions. Export to CSV.

python3 permission_audit.py
🗑️

Privacy Hardening

Disable ad tracking, telemetry, location. 3 hardening levels.

python3 harden.py --level 2
🔍

Privacy Check

Audit current privacy status with score report.

python3 check.py
🎮 Automation & Control
🖥️

Screen Mirror

Mirror device to PC with scrcpy. Multiple profiles: gaming, recording, audio.

python3 launch.py gaming
🎬

Tap/Swipe Automation

Record and replay touch sequences. JSON-based scripts.

python3 automate.py --script actions.json
🔔

Notification Listener

Monitor and auto-reply to notifications with rules.

python3 notif_monitor.py
🛠️ Developer Tools
💉

Frida Scripts

SSL bypass, root detection bypass, crypto logging, intent monitoring.

frida -U -f app -l ssl-bypass.js
🩹

APK Patcher

Patch APKs: disable SSL pinning, bypass root detection.

python3 patch.py --apk app.apk --ssl-bypass --sign
🔋

Battery Analysis

Parse bugreports, find wakelock culprits, export to CSV.

python3 analyze.py bugreport.zip
