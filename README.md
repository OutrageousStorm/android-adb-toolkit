# 🤖 Android ADB Toolkit

Web-based GUI for ADB — control your Android device from a browser without command line.

## Quick Start

```bash
docker-compose up
# Open http://localhost:8000
```

## Features

- Device info viewer
- App manager (install, uninstall, launch)
- File explorer
- ADB shell with autocomplete
- Screenshot capture
- Permission auditor
- Settings viewer/editor

## Manual Start

```bash
pip install flask
python3 server.py
# http://localhost:8000
```
