## 🚀 [→ Open the Live Tool](https://tom-app-a1353ad1.base44.app/ADBTool)

A free, interactive web tool with 50+ ADB commands organized by category. Click any command to copy it instantly.

## Features

- 📱 **Device Info** — model, version, battery, screen specs
- 🧹 **Debloat & Apps** — uninstall/disable bloatware without root
- 🔒 **Privacy Hardening** — revoke camera, mic, location permissions per app
- ⚙️ **System Tweaks** — animations, DPI, wireless ADB, screenshots
- 📁 **File Transfer** — push/pull files, install APKs
- 🛠️ **Bootloader & Recovery** — fastboot, sideload, flash

## Quick Start

```bash
# 1. Enable USB Debugging on your device
# Settings → Developer Options → USB Debugging

# 2. Connect and verify
adb devices

# 3. Use the interactive tool to find and copy commands
# → https://tom-app-a1353ad1.base44.app/ADBTool
```
